List Comprehension vs Generators

Can you explain the difference between a list comprehension and a generator expression in Python? When would you prefer one over the other?

# Example A
squares = [x**2 for x in range(1000)]

# Example B
squares_gen = (x**2 for x in range(1000))
You

List comprehensions create the entire list in memory immediately, while generators produce items one by one (lazily), which is much more memory efficient for large datasets!
